# Rustic Tetris

Tetris with the guideline ruleset and a retro theme.
Written in SDL2 and Rust for fun.

## Building

Requires vcpkg to build.

```bash
cargo install cargo-vcpkg
cargo vcpkg build
cargo build --release
```
